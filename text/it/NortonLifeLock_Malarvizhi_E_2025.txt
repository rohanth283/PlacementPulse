Round 1: Written Coding Test

The first round was a written coding test that included three questions focusing
on arrays and strings:

1. Arrows Alignment Problem: Determine how many arrows should be rotated
to make all of them point in the same direction.

   Input: “^>><>”

   Output: 2

2.  Three  Positive  Integers  Sum:  Find  the  maximum  sum  of  three  positive
integers from the given array.

   Input: {4, -2, 0, 5, -2, 1, 6}

   Output: 10

3. Find Repeating Letter: Identify the  letter that repeats  in a string where all
other characters are distinct.

   Input: "aba"

   Output: ‘a’

Note:  This  required  an  efficient,  in-place  solution  to  minimize  time  and  space
complexity.

Round 2: Technical Interview 1

This  interview  was  focused  on  my  resume,  projects, and  technical  knowledge,
including deep questions on my project details. Key areas covered included:

Data Structures and Algorithms (DSA): A coding question to reverse a linked
list.

Note:  This  required  an  efficient,  in-place  solution  to  minimize  time  and  space

Aptitude:

1)  Write  an  optimized  solution  to  find  the  square  root  of  a  number.

2)  Three humans and three demons are on one side of a river, and they need

a time. The rule is that if the demons outnumber the humans on either side

way to get everyone across the river safely, without ever  letting demons

Other  Technical  Areas:  Questions  on  Object-Oriented  Programming  (OOP),

The second technical interview involved in-depth coding and problem-solving:

Coding Question: Move all zeroes to the end of the array.

   Input: {2, 0, 1, 0, 5, 6}

   Output: {2, 1, 5, 6, 0, 0}

Note:  This  required  an  efficient,  in-place  solution  to  minimize  time  and  space

Other Questions: Write OOP code and SQL code, along with questions from my

about my projects and some coding examples from my work. They also inquired
