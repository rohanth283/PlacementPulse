Question 1 – Dataset Cleaning Cost You are given a string dataset of even length 
consisting of lowercase English letters. You can remove any two characters in one 
operation: 
If both are the same → cost = matchCost
If different → cost = mismatchCost 
Your goal is to remove all characters while minimizing the total cost.
Testcase 1:
dataset = "ouio", matchCost = 2, mismatchCost = 4
Here the matchCost is lower than mismatchcost so need to match as many chars as possible
'o' - 'o' got matched cost=2  'u','i' left in the string
'u' - 'i' got mistmatched so cost=cost+mismatchcost=2+4=6 string is now empty so return 6
Testcase 2:
s = "ouio", matchCost = 6, mismatchCost = 4 
Here the matchcost> mismatchCost so try to mismatch chars as possible to reduce the cost
'o'-'u' cost=4
'i'-'o' cost=cost+mistmatch=4+4=8 so return 8
Testcase 3:
s='ooouio' matchcost=6 mismatchcost=4
Here we have to match a 'o'-'o' no other way so ans would 14
From the question I figured out we are trying out different possiblities and choosing 
the minCost, So figured it as a Dp problem and coded a soln that passed 13/15 Testcases.
