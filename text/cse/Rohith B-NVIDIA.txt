Online Assessment: 
• Duration: 1 hour 
• Platform: HackerRank 
• Pattern: 2 coding questions (one easy and one hard) & 27 MCQs 
including C/C++ fundamentals, OS, DSA, Debug and Aptitude. 
• For the aptitude questions each carries 3 points, for technical 
questions each carries 2 points, the easy question carries 10 or 15 
points and the hard one carries 75 points. 
The MCQs were split into two sections 15 aptitude questions and 12 
technical questions. For the aptitude section focus on permutation, profit 
percent, Chain rule problem and basic math questions. 
 
For the technical questions focus on OS concepts like scheduling 
algorithms, paging, C/C++ fundamentals like string literals, pointers, static 
variables and functions, bit manipulation, signed/unsigned char or int, and 
some DSA analyzes and debug questions.

For the DSA, the easy question should be solved only using C and the hard 
question can be solved using either C/C++ ( C and C++ are the only

The first question is powers of two where a binary char array is provided 
and you have to check whether is it a power of 2.  
( The logic is based on the mathematical property that a positive number is 
a power of two if and only if its binary representation contains exactly one 
'1' ) 
int isPowerOfTwo(const char *str) { 
    int one_count = 0; 
    int i = 0; 
    while (str[i] != '\0') { 
        if (str[i] == '1') { 
            one_count++; 
        } 
        i++; 
    }   
    return (one_count == 1); 
} 
int main() { 
    int n; 
    int i; 
    char binary_string[101]; 
    scanf("%d", &n); 
    for (i = 0; i < n; i++) { 
        scanf("%100s", binary_string); 
        if (isPowerOfTwo(binary_string)) { 
            printf("True\n"); 
        } else { 
            printf("False\n"); 
        } 
    } 
    return 0; 
}
