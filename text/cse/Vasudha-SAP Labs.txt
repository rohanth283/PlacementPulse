ONLINE TEST :
This round consists of two coding questions with a duration of 45 min .
1.
Input : An unsorted array and an integer k. (k >=1)
Output : Print the kth smallest element in the unsorted array.
K’th Smallest / Largest Element in Unsorted Array
2.
Input : An integer array of length n with elements 0 to n-1.
Output : Print an array where the elements have to be rearranged, where the elements are
taken as indices and the indices as elements .
Example :
Input :  [3, 0, 1, 2]
Output : [1, 2, 3, 0]
Let a be the input array .
a[0] = 3 ; a[1] = 0 ; a[2] = 1 ; a[3] = 2
Output should be :
a[3] = 0;  a[0] = 1 ; a[1] = 2 ; a[2] = 3
