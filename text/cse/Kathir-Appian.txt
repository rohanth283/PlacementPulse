The coding questions (Java was the only language available) were
1.A easy problem involving HashMap. (Leetcode easy level)
2.Breadcrumbs in a website.
Given an ArrayList<ArrayList<String>>navigations generate the actual navigation tree.
Example
[
[“Root”, “Profile”, “Settings”],
[“Root”, “Profile”, “Albums”],
[“Root”, “Settings”],
]
Helper class TreeNode was given
class TreeNode {
string value;
ArrayList<TreeNode>children;
void addChild(TreeNode c){
this.children.add(c);
}
//a function to print the tree
}
The function that you need to implement is
void createTree (ArrayList<ArrayList<String>> paths) {
}
The solution is
import java.util.*;
class TreeNode{
string value;
ArrayList<TreeNode>children;
TreeNode(string str) {
this.value = str;
}
void addChild (TreeNode c) {
this.children.add(c);
}
}
class answer{
void createTree(ArrayList<ArrayList<String>> paths){
TreeNode parent = new TreeNode("Root");
HashMap<String, TreeNode> map = new HashMap<>();
for (ArrayList<String>path : paths) {
for (int i = 1 ; i < path.size(); i++) {
boolean alreadyIncluded = false;
String parent = path.get(i-1);
String current = path.get(i);
for (TreeNode child : parent.children){
if (child.value == current){
alreadyIncluded = true;
break;
}
}
if (!alreadyIncluded) {
TreeNode child = null;
if (!map.containsKey(current)){
child = new TreeNode(current);
map.put(current, child);
}
else{
child = map.get(current);
}
parent.addChild(child);
}
}
}
}
}

First Technical Round
1.A coding question (Find the majority element in the array)
https://leetcode.com/problems/majority-element/.
You have to solve the problem in constant space and in linear time. Use the Moore’s voting
algorithm(https://www.geeksforgeeks.org/boyer-moore-majority-voting-algorithm). You have
to actually code out the solution in a editor and pass the test cases.Then a follow up
question on how to find the element that occurs more than 40% of the time in the array was
asked.The constraint given was constant space. Sorting the array taking the count would
work. Time given for the coding question was 20 minutes. Then resume related questions
were asked.As I had mentioned React JS in my skills questions related to React was asked.
1. Advantages of using React
2.What is prop drilling.
3.UseContext hook.
4.What is JSX.
5.Advantages of using JSX over traditional JS.
Difficulties faced during the project. I mentioned how deploying the project was difficult as
the server cache was less. So they asked about the deployment experience in azure and
