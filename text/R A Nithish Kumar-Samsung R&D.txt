The question was something like:  
There is an array which has memory spaces allocated to it, 
when a program utilizes the CPU the spaces in the array are 
allocated to each program from the array. Write an algo to 
optimize the memory allocation efficiently.  
Sol: Use a Boolean variable ,Mark the used index(Utilized 
spaces) of array as false and keep track of free spaces and 
check for adjacent index which is also free and expand the 
available free places to make it as a big chunk of available 
memory to avoid segmentation .
