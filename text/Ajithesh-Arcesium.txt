Section 3 : Coding – 2 questions (45 mins) 
1. Naruto Clan Curse – It is an application of level-order traversal on binary tree, where we 
have to return the avg of all the unaffected node values from each level. Min value of the 
extreme nodes of each level is calculated. A node is affected when its value is less than the 
above min value.  
2. ATM Change problem – It is a dynamic programming problem. A req amount, preferred 
denomination and set of available denominations are given. We have to return the number 
of  ways to make change. 
Ex : Amount = 1200  Preferred denomination = 500  denominations = [ 100, 200, 500] 
1. 1200 = 500 + 500 + 100 + 100 
2. 1200 = 500 + 500 + 200 
Output : 2 
 
Solution : Modification of  https://www.geeksforgeeks.org/coin-change-dp-7/ 
 
Technical Round 1 : (1 hour) 
This round was a pair coding round. 
I was given a backtracking question. First asked to write the pseudo code and optimize it. At last, 
code it in the language of your choice. 
Question : Given n x n matrix filled with 1, 2, 3 . Find the path with max reward from src to dest. 
1 -> move one cell right 
2 -> move one cell down 
3 -> movie one step either right or down 
1 2 1 2 
3 3 1 2 
1 2 1 2 
3 1 1 2  
Src = (0, 0) 
Dest = (3, 3) 
Max reward path = 1 -> 2 -> 3 -> 1 -> 2 -> 2 -> 2 = 13

Technical Round 3 : (1 hour) 
I was given few coding questions and come up with an algorithm with specified time and space 
complexities. 
Note : Discuss all possible edge cases and efficiency of the algorithm in each case. 
1. Given an array of n consecutive integers where one of the element is over-written by 
another element in the array. Now this array is shuffled and find the element being over-
written with minimum time as possible without any auxiliary space. 
Ex: A[] = [51, 52, 53, 54, 55] -> [51, 52, 53, 52, 55] -> [52, 55, 51, 53, 52] 
      Result = 54 , as 52 over-writes 54 
 
2. Given a sorted array where all elements except one element is repeated twice. Find the 
distinct element with minimum time and O(1) space. 
Ex : A[] = [1, 1, 2, 2, 3, 4, 4, 5, 5]  Result = 3 
 
3. Given an mathematical expression place the parenthesis such that value of the expression 
is maximum. 
Ex : 5*2-4/1+3 
Ref : https://www.geeksforgeeks.org/all-ways-to-add-parenthesis-for-evaluation/
